Version 3.0

january 5, 2008 - Version 3.1.0

  • operators +, -, *, / on volumes
  • improved pyhton bindings (pyaims) accordingly
  • VolumeUtil class providing apply and selfApply functions that operate a function on every voxel of one or two volume(s)
  • IO classes on VolumeRef
  • Improvements in Volume API
  • Optional use of the Blitz++ library. When present, blitz++ is used to implement volume accessors and iterators, which both dramatically improves performance and will allow the use of strides (array ordering).
  • "normalized" some volume header properties: "referentials", "transformations", "storage_to_memory" allow to completely handle image orientation
  • python bindings for most of cartodata library: see the pyaims project
  • other little things...
